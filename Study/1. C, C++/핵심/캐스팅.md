## static_cast

C언어 강제형변환의 문제점을 줄인 방식이다.

기본적으로 강제 변환이 아닌 논리적으로 변환이 가능한 타입에 대해서만 캐스팅을 할 수 있고, 논리적으로 맞지 않다면 컴파일 에러가 발생한다.
```cpp
const int num = 10;
int* p = NULL;
p = static_cast<int*>(&num); // 컴파일 에러
```

## dynamic_cast

안전하게 다운캐스팅을 할 때 가장 많이 활용되고, 하나 이상의 가상함수가 있는 상속관계에서 사용된다.
```cpp
class Parent {
	virtual void func() { }
};

class Child : public Parent {
	void func() override {}
};

int main() {
	Parent* p = new Child;
	Child* c = dynamic_cast<Child*>(p);
}
```
